# 🌎 Geo-Referenciados 711

API de geolocalización con Node.js. Manejo de coordenadas, distancias y puntos de referencia.

## 🚀 Quick Start

```bash
git clone https://github.com/FrankAboytes/Geo-Referenciados711.git
cd Geo-Referenciados711
npm install
npm start
```

## 🛠️ Stack

- Node.js + Express
- MongoDB / Mongoose
- API REST geoespacial
